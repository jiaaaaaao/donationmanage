This program is intended for terminal interaction.
1. Create a donation
   register:<User>:<Donation Type>:<Quantity>:<Date>
   e.g.
   
   register:jiao:cash:100:2024
2. Create a distribution
   register:<Donation Type>:<Quantity>:<Date>

   e.g.
   register:cash:50:2024
3. Create report
   option1:
   report:inventory
   option2:
   report:donor
